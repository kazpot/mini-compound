# Mini Compound

Minimum version of Compound protocol (only for test)

## Installation

1. Install

```
$ npm install -g yarn
```

2. Create .env file

```
ALCHEMY_API_KEY={alchemy api key}
WALLET_KEY={wallet mnemonic words}
```

3. Compile smart contracts

```
$ yarn build
```

4. Deploy to rinkeby testnet

```
$ yarn deploy:rinkeby
```
